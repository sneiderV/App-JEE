--> Iteracion 2 
Para el despliegue de la aplicacion debe estar instalado Eclipse Neon JEE con java 8.0 y
JDK 8.0 , ademas por problemas presentados en la infraestructra de la universidad para
la fecha de entrega del proyecto la base de datos debe ser LOCAL, se utilizo Oracle Express
con las corresondientes credneciales vistas en el archivo .properties, finalmente se 
requiere de un servidor Wifly x 10.0.
Se adjuntan dos colecciones en formato postman, uno con las pruebas de Requerimientos funcionales
y otro con las pruebas para os requerimientos funcionales de consultas.
